# Mosaic Documentation

Welcome to the official documentation for **Mosaic**—a platform for creative communities to build together, show up together, and share value.

Mosaic is designed to shift the focus from individual social feeds to shared workspaces (called Villages). By using database records and Cardano blockchain verification, Mosaic enables trust and community ownership to grow.
